How to Use This App

Plain-English version. No software goblin dialect required.
This app has one job: help you capture work as it happens so your Excel time sheet is not built from vibes, fragments, and courtroom-adjacent guesswork.

1) Add your matters first

  1. Open Matters.
  2. Add each client or matter you work on.
  3. Pick a color for each one so lists are easier to scan.
  4. Save the matter. After that, it shows up everywhere else in the app.

2) Use Quick Add for short finished work

  1. On Dashboard, choose a task type.
  2. Select the matter.
  3. Write a short description.
  4. Enter the minutes.
  5. Save the entry.
Best for short emails, short calls, quick reviews, and tasks you already finished.

3) Use the timer for longer work

  1. Choose the matter, task type, and description in the timer section.
  2. Click Start Timer.
  3. When the work changes, click Pause or Stop & Save.
  4. If you click Start Timer again for new work, the app pauses the old timer automatically.
That means you can switch clients without losing the first timer. Tiny miracle. Useful one, too.

4) How paused timers work

  1. A paused timer stays in the Paused Timers list.
  2. You can Resume it, Save to Log it, Load to Form, or Discard it.
  3. If you return to that client later, resume the paused timer instead of starting from scratch.
  4. The app also warns you when paused timers are sitting there waiting to be dealt with.

5) How to make a preset

  1. Fill out the Quick Add form or the Timer form with a common setup.
  2. Click Save Current as Preset or Save Timer Setup as Preset.
  3. Give it a simple name you will recognize fast.
  4. Examples: Smith client email, Jones drafting, Court follow-up call.
  5. Favorite the presets you use constantly so they stay at the top.

6) What a preset actually does

  1. Fill Entry loads the preset into Quick Add.
  2. Load Timer loads the preset into the timer form.
  3. Start Timer loads the preset and starts timing right away.
  4. Presets save clicks. They do not save completed work by themselves.
A preset is a reusable setup, not a finished time entry. It is the difference between a template and an actual billable line.

7) Review and export

  1. Open Review & Export.
  2. Pick the date you want to review.
  3. Check the entries.
  4. Click Export for Excel.
Use the filter bar to narrow the list by matter/client or search words before export. Every exported row includes service type, price type, set price, and line total so Excel can sort it cleanly. The Weekly View also helps you catch weird gaps before export.

8) Reuse or duplicate past work fast

  1. On Today's Timeline, Review & Export, or Weekly View, click Reuse to load that entry back into Quick Add.
  2. Click Duplicate to create a new copy dated today with the same setup.
  3. Reuse is better when you want to tweak the description or minutes first.
  4. Duplicate is better when the work line is basically the same and you just need it logged again.
This is the anti-retyping feature. Let the app remember the pattern so your hands do less nonsense.

9) Use Recent Entry Shortcuts for repeat work

  1. On the Dashboard, look for Recent Entry Shortcuts.
  2. These are built from finished entries you already logged.
  3. Click Duplicate Today to log that same type of work again right away.
  4. Click Load to Quick Add if you want to tweak the text or minutes first.
  5. Click Pin to keep your most-used shortcuts at the top.
Think of this strip as a fast lane for repeat work lines. It is pulled from real past entries, not from presets you still have to finish filling out. Use the Show menu to switch between Smart Now, Most Used, Top Used This Week, or Most Recent, and use Visible to decide how many shortcut cards stay on screen. When you pick a matter in Quick Add or the timer, the strip automatically narrows to that client when it has matching history.

10) Use matter totals before export

  1. Open Review & Export for one day or Weekly View for the week.
  2. Check the Matter Totals table to see totals by client.
  3. Use Filter This Matter if you want to narrow the screen to one client fast.
  4. Export after the totals and entries look right.
This catches missing work and weird pricing sooner. Excel should be the final destination, not the first place you discover the day fell apart.
